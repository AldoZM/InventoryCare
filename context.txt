== TECH STACK DETAIL ==
- Backend: FastAPI + SQLite (migrated FROM PostgreSQL)
- DB connection: sqlite3 + threading.Lock (no pool)
- Auth: JWT tokens
- Frontend: vanilla HTML/CSS/JS in /www folder
- Launcher: launcher.py — pystray system tray icon, opens browser, runs uvicorn HTTPS
- Build: PyInstaller (inventarycare.spec) → one-folder bundle
- Installer: Inno Setup (installer.iss) → .exe installer for Windows

== HOW TO REBUILD .EXE ==
ALWAYS kill process first or dist\ will be locked:
  Stop-Process -Name "InventaryCare" -Force -ErrorAction SilentlyContinue
  python -m PyInstaller inventarycare.spec --noconfirm
Output: dist\InventaryCare\InventaryCare.exe
NEVER use --distpath workaround. Kill first, build to dist\ directly.

== IMPORTANT DECISIONS ==
- SQLite not PostgreSQL: target = non-technical users, no server setup
- One-folder PyInstaller (not onefile): faster startup, easier debug
- Vanilla JS: no React/Vue, keep simple for small app
- pystray: needed for Windows tray icon without extra deps
- html5-qrcode bundled local (not CDN): offline use in installer
- qrcode.min.js bundled local: offline QR generation in settings
- pytesseract optional: app works without it, OCR server returns 503 gracefully
- Tesseract.js loaded from CDN lazily (OCR tab only): requires internet for client OCR
- HTTPS self-signed cert: stored in AppData/InventaryCare/, regenerated if expired or IP changed
- Login uses window.location.reload() + sessionStorage flag (not navigate()) to ensure showApp() runs
