== DISEÑO APROBADO: HTTPS LAN ACCESS (pendiente implementar) ==
Objetivo: celulares en la misma red WiFi puedan usar la cámara para escanear códigos.
getUserMedia requiere HTTPS en IPs no-localhost — sin HTTPS el botón 📷 no funciona en cel.

Decisión: UN solo server HTTPS en 0.0.0.0:8080
- PC abre https://localhost:8080 (aviso cert una vez, luego ok)
- Celular abre https://192.168.x.x:8080 (aviso cert una vez, luego ok)
- Cámara funciona en celular después de aceptar cert

Cambios necesarios en launcher.py:
- Generar cert SSL auto-firmado en primer arranque (cryptography lib, ya instalada via python-jose)
- Guardar cert.pem + key.pem en AppData/InventaryCare/
- Regenerar si expirado (válido 1 año)
- Detectar IP local con socket stdlib
- Pasar ssl_certfile + ssl_keyfile a uvicorn.run()
- Agregar item en tray: "Red local: https://192.168.x.x:8080"

Cambios en inventarycare.spec:
- Agregar hidden imports de cryptography (hazmat.primitives, x509, backends, etc.)

NOTA: aviso "Your connection is not private" es esperado y normal con cert auto-firmado.
Trabajadores aceptan UNA vez en su cel, después no vuelve a aparecer.

== HOW TO RUN (dev) ==
cd InventaryCare
venv\Scripts\activate
uvicorn app.main:app --reload
Open http://localhost:8000

== HOW TO RUN (launcher) ==
python launcher.py
(opens browser + tray icon)

== HOW TO REBUILD .EXE ==
ALWAYS kill process first or dist\ will be locked:
  Stop-Process -Name "InventaryCare" -Force -ErrorAction SilentlyContinue
  python -m PyInstaller inventarycare.spec --noconfirm
Output: dist\InventaryCare\InventaryCare.exe
NEVER use --distpath workaround. Kill first, build to dist\ directly.

== IMPORTANT DECISIONS ==
- SQLite not PostgreSQL: target = non-technical users, no server setup
- One-folder PyInstaller (not onefile): faster startup, easier debug
- Vanilla JS: no React/Vue, keep simple for small app
- pystray: needed for Windows tray icon without extra deps
- html5-qrcode bundled local (not CDN): offline use in installer
- pytesseract optional: app works without it, OCR server returns 503 gracefully
- Tesseract.js loaded from CDN lazily (OCR tab only): requires internet for client OCR
